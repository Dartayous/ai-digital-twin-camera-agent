# Project 07 Scope

## Flagship Scenario
Autonomous inspection robot patrols a smart interior environment, detects a target condition, and changes behavior in response.

## What this project WILL include
- One authored indoor digital twin environment
- One Isaac Sim robot/agent
- At least one working sensor
- One observable perception signal
- One explicit decision state
- One resulting action change
- One clear demo path from normal behavior -> detected condition -> changed behavior

## What this project will NOT include in v01
- Full ML training pipeline
- Multi-robot coordination
- Cloud services
- Complex dashboard stack
- Cosmos runtime dependency

## Success Condition
A reviewer can watch the demo and clearly see:
1. baseline behavior
2. sensor-driven detection
3. decision transition
4. action change
